# Architecture Decision Records

This directory holds megálos's ADRs. One decision per file. Numbered sequentially (`NNN-short-slug.md`), starting at `001`. Once an ADR is accepted it is not edited in place; subsequent ADRs supersede or amend it and reference it by number.

Each ADR has a status field (Draft, Accepted, Superseded by ADR-NNN, Deprecated), a date, a Context section, a Decision section, a Consequences section, and a "what would have to be true for this to be wrong" section. Match the structure and register of the existing ADRs when adding a new one.

ADRs are tier-3 binding context per `2026-04-22-operator-handbook.md` §8 — load the relevant ADR into the context pack for any work that touches its decision surface.
